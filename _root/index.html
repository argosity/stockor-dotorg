Stockor is now Free Software ...

Argosity is in the processes of open-sourcing their crown jewel, Stockor.

TL;DR: Wondering what Stockor ERP is? Visit Stockor.com or jump to the Demo.

Stockor is a ERP system that contains all the tools you'll need to run a small to medium sized business on the internet. We're slowing extracting the code from our monolithic Rails app and splitting it into separate gems and engines.

Follow our progress below. Learn more about Stockor and Argosity

Want to see the gauge go up faster? Got a business issue that needs addressed TODAY? Let us know

Plugins; Mailing Lists, Barcoding, iOS Point of Sale app, & Amazon Integration.

As part of the open-sourcing effort, we're going to split off anything that's not core into plugins. We'll also going to make sure they are well documented and provide a clear example of how to extend Stockor.

E-commerce Website. Built in coffee-script using Backbonejs with Bootstrap UI. [stockor.com demo]

Workspace a Rails Engine serving the Ext JS application. Consists of a Tab-based UI with around 50 screens. [stockor.com demo]

API: a Rails Engine with ~ 60 REST endpoints. It's a thin JSON layer over the Stockor core gem.
